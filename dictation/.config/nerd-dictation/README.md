# Nerd Dictation

`setup-nerd-dictation` manages the executable and model directories under this path.

Expected model layout:

- `~/.config/nerd-dictation/models/en`
- `~/.config/nerd-dictation/models/pt`
- `~/.config/nerd-dictation/models/de`

`~/.config/nerd-dictation/model` remains the default English-compatible path.
